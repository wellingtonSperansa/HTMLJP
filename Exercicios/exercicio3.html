1. Classes (class)
Uso: Classes são usadas para aplicar estilos ou comportamentos a múltiplos elementos.
Boas práticas:
Use nomes descritivos que indiquem a função ou o propósito do elemento (ex.: menu-principal, botao-enviar).
Utilize o padrão kebab-case (palavras separadas por hífens), como caixa-de-texto.
Evite caracteres especiais, espaços ou números no início do nome.
Prefira nomes genéricos para reutilização (ex.: card, titulo, container).
2. IDs (id)
Uso: IDs são únicos e devem ser aplicados a apenas um elemento por página.
Boas práticas:
Use nomes únicos e específicos (ex.: formulario-contato, cabecalho-site).
Utilize o padrão camelCase (primeira palavra minúscula, palavras seguintes com inicial maiúscula), como
menuPrincipal.
Evite reutilizar o mesmo ID em diferentes elementos.
Não use IDs para estilização, a menos que seja absolutamente necessário (prefira classes).
